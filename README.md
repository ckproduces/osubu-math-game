# Osubu
Osubu basit bir matematik oyunudur.
Osubu'da yapman gereken, sayı dizisinin kuralını tahmin etmek!
**Kurallar basit:**
- Osubu'da 4 adet operatör bulunur: toplama (+), çıkarma (-), çarpma (*) ve üs alma (^)
- Cevabını 4* veya ^3 şeklinde yazmalısın.
- Osubu'da toplam 10 seviye bulunur. 10 seviyeyi tamamladığında oyunu bitirmiş olursun.
